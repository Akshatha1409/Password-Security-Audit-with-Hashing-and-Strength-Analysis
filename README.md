# Password Security Audit Tool

This project demonstrates a simple password security audit by validating password strength and securely hashing it using SHA-256.
This tool highlights common weaknesses and shows how passwords should be stored securely.

## Features
- Password strength validation
- Secure hashing using SHA-256
- Security best-practice recommendations

## Technologies Used
- Python
- Regular Expressions
- Cryptographic Hashing
